## Looping code

### Looping through a collection

#### The for...of loop

The basic tool for looping through a collection is the `for...of` loop:

```javascript
const cats = ['Leopard', 'Serval', 'Jaguar', 'Tiger', 'Caracal', 'Lion']

for (const cat of cats) {
    console.log(cat);
}
```
